## 项目介绍

结合了以下两个命令
基于 Egg + React + Webpack 的前后端分离项目

- Egg 版本： ^2.x.x
- Node 版本: Node ^10.x.x+
- Webpack 版本: ^4.30.0
- React 版本: ^16.0.0


###使用

####安装依赖

```shell
yarn
```

#### 本地开发

```bash
yarn dev
```

####启动项目

```
yarn start
```

#### 项目构建

```bash
yarn build
```
